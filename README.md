# go_restart
Повторяю материал пройденный на курсах

# 1. Разогрев: переменные, типы, условия

1. Объявить переменные всех основных типов: int, float64, string, bool, rune.
2. Написать программу, которая читает число и определяет, чётное оно или нет.
3. Сделать конвертер температур: Celsius ↔ Fahrenheit.
4. Написать калькулятор на if / else и switch.
5. Найти максимум из 3, 5 и 10 чисел.
6. Проверить, является ли строка палиндромом.
7. Подсчитать количество гласных, согласных, цифр и пробелов в строке.

# 2. Циклы и базовая работа со строками/массивами
8. Вывести таблицу умножения.
9. Суммировать числа от 1 до N.
10. Найти факториал.
11. Сгенерировать массив чисел и найти min/max/среднее.
12. Развернуть массив вручную, без встроенных функций.
13. Удалить дубликаты из слайса.
14. Отсортировать слайс пузырьком, потом встроенной sort.
15. Подсчитать частоту каждого элемента в слайсе.

# 3. Функции
16. Написать функции с параметрами и возвращаемыми значениями.
17. Сделать несколько версий одной функции: обычная, с несколькими return, с variadic-параметрами.
18. Написать функцию, которая принимает другую функцию как аргумент.
19. Реализовать map, filter, reduce для слайса.
20. Сделать пакет с набором математических или строковых утилит.

# 4. Указатели и память
21. Понять разницу между передачей по значению и по указателю.
22. Написать функцию, которая меняет значение переменной через указатель.
23. Сделать структуру User и функции, которые обновляют её поля через указатель.
24. Понять, когда Go сам берёт адрес, а когда разыменовывает.
25. Создать слайс и понаблюдать, как меняется len и cap при append.
26. Написать функцию, которая принимает слайс и изменяет его элементы.
27. Разобраться, почему append иногда создаёт новый массив.

# 5. Struct, методы, интерфейсы
28. Создать несколько структур: User, Order, Product.
29. Добавить методы к структурам: String(), IsValid(), UpdateName().
30. Сделать композицию структур через встраивание.
31. Написать интерфейс Writer/Reader и свою реализацию.
32. Понять, чем отличается метод с value receiver от pointer receiver.
33. Сделать мини-проект: сущности Task, Project, Employee и методы работы с ними.

# 6. Карты, ошибки, JSON
34. Посчитать статистику слов в тексте через map[string]int.
35. Реализовать кэш в памяти на map.
36. Создать свои ошибки через errors.New и fmt.Errorf.
37. Сделать функцию, которая валидирует данные и возвращает ошибку.
38. Сериализовать структуру в JSON и обратно.
39. Обработать JSON из файла или из строки.
40. Написать простой конфиг-загрузчик из JSON/YAML.

# 7. Работа с файлами и CLI
41. Прочитать файл построчно.
42. Записать данные в файл.
43. Сделать копирование файла.
44. Написать CLI-утилиту: подсчёт строк, слов и символов в файле.
45. Сделать логирование в файл.
46. Реализовать обработку аргументов командной строки.

# 8. Конкурентность
47. Запустить несколько goroutine и дождаться их через sync.WaitGroup.
48. Передать данные через channels.
49. Сделать worker pool.
50. Понять deadlock и устранить его.
51. Ограничить число одновременно работающих goroutine.
52. Использовать context для отмены работы.
53. Написать конкурентный парсер нескольких URL.

# 9. Реальные рабочие задачи
54. Поднять HTTP-сервер на net/http.
55. Сделать CRUD для сущности в памяти.
56. Подключить PostgreSQL и реализовать CRUD уже через БД.
57. Сделать REST API с валидацией, ошибками и логированием.
58. Написать middleware: логирование, recovery, auth.
59. Добавить тесты на handlers, service и repository.
60. Реализовать простой проект: заметки, задачи, пользователи, заявки.

# 10. Уровень ближе к работе
61. Разделить проект на слои: handler / service / repository.
62. Сделать миграции БД.
63. Добавить конфиг через env и .env.
64. Реализовать пагинацию, фильтрацию, сортировку.
65. Добавить JWT-auth.
66. Написать интеграционные тесты.
67. Собрать Docker-образ и поднять проект через docker-compose.
68. Сделать мини-сервис с очередью, фоновыми задачами и логированием.
